# Отклик

Мобильное приложение для организации взаимопомощи: социальные работники публикуют задачи, волонтёры откликаются и помогают людям в реальном времени.

## Цель приложения

Приложение помогает быстро организовывать поддержку людям, которые в ней нуждаются.  
Социальные работники создают объявления, волонтёры откликаются и выполняют задачи.

## Роли пользователей

### Социальные работники (Координаторы)
- Создают объявления о помощи
- Указывают: название, описание задачи, дату окончания
- Просматривают заявки волонтёров
- Принимают или отклоняют отклики
- Могут посмотреть профиль волонтёра перед принятием решения
- Редактируют свой профиль

### Волонтёры
- Регистрируются в приложении
- Просматривают актуальные объявления
- Подают заявку (отклик) на интересующую задачу
- Видят статус своих заявок (ожидает, принято, отклонено)
- Могут отказаться от участия
- Редактируют свой профиль

**Обе роли могут:**
- Сортировать и фильтровать объявления
- Оставлять комментарии под объявлениями

## Технологии

- **Expo** (managed workflow)
- **React Native**
- **Expo Router** (file-based routing)
- **TypeScript** — полная типизация
- Управление состоянием:  Zustand 
- Стили: StyleSheet
